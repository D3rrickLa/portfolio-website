# 🌐 Portfolio Website
A fast, modern, and minimal personal portfolio built to showcase my projects, experience, and everything I’m working on as a developer.

This site is designed to be clean, responsive, and easy to maintain — powered by Astro, TypeScript, and Bun for a smooth developer experience.

## 🚀 Tech Stack
- Astro - Lightning‑fast static site builder

- TypeScript - Type‑safe development

- Bun - Ultra‑fast runtime & package manager

- Modern CSS - Component‑driven styling

- GitHub Pages - Hosting & deployment

## 📁 Project Structure
Cod

### 🛠️ Development
Run these commands from the project root:

| Command | Description | 
|---------|----------|
| bun install	| Install dependencies |
| bun dev | Start local dev server (```localhost:4321```) |
| bun build	| Build production output to ```/dist``` |
| bun preview | Preview the production build |
| bun astro ```...```	| Run Astro CLI commands |
